## Features

- Ethereum-compatible JSON-RPC endpoint
- Automatic contract deployment
- Data persistence via Docker volumes
- Lightweight (~80-120MB RAM)

## Endpoints

- RPC: `http://localhost:8545`
- Chain ID: 1337 (default)
